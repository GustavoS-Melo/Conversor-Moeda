# Currency Converter - Real Time API

Um conversor de moedas moderno e responsivo que utiliza taxas de câmbio em tempo real para fornecer conversões precisas entre diversas moedas globais.

##  Link de Acesso
O projeto está hospedado e pode ser acessado em:  
**[https://conversor-de-moedaa.netlify.app](https://conversor-de-moedaa.netlify.app)**


##  Funcionalidades
- **Cotações em Tempo Real**: Integração com a ExchangeRate-API.
- **Design Moderno**: Interface baseada em Glassmorphism e CSS Variables.
- **Assincronismo**: Uso de `async/await` para manipulação de requisições de rede.
- **Tratamento de Erros**: Feedback visual ao usuário em caso de falhas de conexão ou inputs inválidos.

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização avançada, Flexbox e Variáveis.
- **JavaScript (ES6+)**: Lógica de conversão, manipulação de DOM e Fetch API.
- **Netlify**: Deploy e hospedagem contínua.
- **ExchangeRate-API**: Fonte de dados financeira.
